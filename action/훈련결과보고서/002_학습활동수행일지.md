--- 
### 11월 18일 (화) 

### 수행훈련내용 (총 6시간)
- [AI의 이해] LLM 연계 전 LLM의 작동 원리 및 본질을 이해하기 위한 이론적인 내용 학습
- [AI의 이해] LLM의 특성으로 인해 기존 개발과 LLM 연계 개발의 프로세스 차이 이해
- [AI의 이해] LLM 개발이 기존 개발과 다르게 갖는 한계점 이해
- [AI의 이해] 한계점에 대한 실질적인 실습을 위한 환경구성 (Ollama 설치)
- [AI의 이해] LLM의 한계점을 실제적으로 이해하기 위한 Ollama 설치 API 사용 실습
- [AI 한계 보완] LLM의 한계점을 보완하는 시스템적 보완 기법 소개 (프롬프트 엔지니어링, Chat Memory, RAG, Tool Calling ... )
- [AI 한계 보완] 로컬 모델 선정을 위한 Ollama 설치 모델 사용 및 식별 방법 소개
