# 서비스 분석 
 
> [ 서비스명 ]  Re-Turn

> [ 서비스 개요 ]   
> 이 서비스는 경도인지장애(Mild Cognitive Impairment, MCI) 사용자가 자신의 기억을 자연스럽게 떠올리고
> 유지할 수 있도록 돕는 디지털 회상치료(Reminis­cence Therapy) 기반 플랫폼이다.

> [ 대표 기능 ]   
> 사용자별 사진그룹화 등록 기능  
> 사진그룹기반 AI 회상 대화 기능  
> 대화 내용을 토대로한 회상 컨텐츠 (이야기 + 이미지)생성 기능   

> [ 서비스 특징 ]  
> 현재 모든 서비스 곳곳에 LLM 연계 기술이 이미 적용되어 있는 서비스  
> 서비스의 연계 곳곳에 프롬프트 요소가 다수  
> 해시태그 분류 기능의 벡터화된 데이터 베이스가 활용되어 있음  

> [ 개선내용 도출 ]   
> 현재 LLM 연계를 통해 AI 와 대화하는 형태의 회상 대화 서비스 제공  
> AI와의 대화가 다소 딱딱하고 친근하지 못함.   
> 사용자가 친숙하게 느낄만한 대상과의 대화처럼 유도하는 방향 검토
